Respostas:

1. Qual a diferença entre os métodos GET e POST em formulários HTML?

GET: Parâmetros passados no cabeçalho da requisição. São usados para solicitar dados ao servidor ou de um recurso.
POST: Envia informações para o corpo da requisição (o servidor), é mais utilizado em formulários.

2. Como o Flask diferencia o tratamento de requisições GET e POST em uma rota?

O método é definido pela aplicação quando se cria uma rota. Caso for POST, é para enviar informações, e caso for GET, é para puxar informações.

3. O que é o objeto request no Flask e para que serve?

Para pegar os dados enviados pelo usuário
